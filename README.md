## 🌟 Overview

**Khata App** is a powerful, feature-rich **Digital Business Ledger & Management System** built with **Flutter**. It replaces the traditional paper-based *بہی کھاتہ* (business ledger) used by traders and shopkeepers with a modern, fast, and secure mobile application.

Designed with **full Urdu/RTL support**, the app provides an intuitive interface for managing every aspect of business operations — from customer ledgers and supplier management to invoicing, stock control, and financial reporting.

> 💡 **Built for real-world use** — This app is production-ready and actively used by businesses in the **UAE & Pakistan**.

## 🏗️ Architecture

The project follows **Clean Architecture** with **Domain-Driven Design (DDD)** principles:

```
lib/
├── main.dart                    # App entry point
├── app_router.dart              # GoRouter navigation
├── firebase_options.dart        # Firebase config (gitignored)
├── core/
│   ├── constants/               # Colors, strings, text styles
│   ├── theme/                   # App theme configuration
│   ├── utils/                   # Currency formatter, date utils
│   ├── widgets/                 # Shared widgets (PDF bottom sheet)
│   └── services/                # Firebase sync service
└── features/
    ├── auth/                    # Authentication (Firebase)
    ├── onboarding/              # Splash & onboarding screens
    ├── home/                    # Dashboard & navigation
    ├── business/                # Multi-business management
    │   ├── data/
    │   │   ├── models/          # Hive models
    │   │   └── repositories/    # Repository implementations
    │   ├── domain/
    │   │   ├── entities/        # Business entities
    │   │   └── repositories/    # Repository interfaces
    │   └── presentation/
    │       ├── providers/       # Riverpod providers
    │       ├── screens/         # UI screens
    │       └── widgets/         # Feature-specific widgets
    ├── customers/               # Customer & Supplier management
    ├── transactions/            # Give/Receive & Party ledger
    ├── cashbook/                # Cash in/out tracking
    ├── stock/                   # Inventory management
    ├── invoice/                 # Billing & invoicing
    ├── register/                # Purchase, Sales & Expense registers
    │   └── presentation/pdf/   # PDF generation for all registers
    ├── reports/                 # Business reports & analytics
    └── settings/                # App settings & data management
```

## 🚀 Getting Started

### Prerequisites

- Flutter SDK `>= 3.0.0`
- Dart SDK `>= 3.0.0`
- Android Studio / VS Code
- Firebase project (for auth & cloud sync)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/muhammadwasif12/Khata-App.git
   cd Khata-App
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Configure Firebase**
   - Create a Firebase project at [Firebase Console](https://console.firebase.google.com/)
   - Enable **Email/Password Authentication**
   - Enable **Cloud Firestore**
   - Download and add your `google-services.json` (Android) and `GoogleService-Info.plist` (iOS)
   - Generate `firebase_options.dart` using FlutterFire CLI:
     ```bash
     flutterfire configure
     ```

4. **Run the app**
   ```bash
   flutter run
   ```

## 🔒 Security

This project takes security seriously:

- 🔐 **Firebase Authentication** for secure user login
- ☁️ **Firestore Security Rules** for data isolation
- 🚫 **Sensitive files excluded** from version control:
  - `firebase_options.dart`
  - `google-services.json` / `GoogleService-Info.plist`
  - `*.jks` / `*.keystore` (signing keys)
  - `key.properties` / `local.properties`
  - `.env` files
